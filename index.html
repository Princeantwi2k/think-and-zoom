Kenti Game

  • Full name: "Prince Antwi"
  • Name Origin : Kwahu Bepong,Ghana
  • Game Designer : Zuby Onwuta
  • Name provided by : Zome Onabamiro
  • Language : Swift

kenti an IOS single player game, designed and developed in under 3 days, is a winner of the Apple Worldwide Developer conference 2015 Minority Scholarship.The objective of the game is free "Kenti", a bird trapped in a cage so she can join her friends.The game is played in 2 modes

A)Hand on with double taps

B)Hands free with an EEG reader such as Neurosky Moblies headset

The game is part of a larger project aimed at using game to conduct on the "Sustained use of EEG in daily life" Work is in progress to develop an app for this research using IOS ResearchKit.
